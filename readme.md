### Student Management Application

1. Create `StudentsDB`
2. Execute `StudentsDB.sql`
3. Start `IdentyServer`
4. Start `Student.API`
5. Navigate to `Student.Client`
     - execute `npm install --legacy-peer-deps`
	 - execute `npm run dev`
	 - start `Student.Client`
